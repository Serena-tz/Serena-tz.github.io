# 在 vue 中使用过渡和动画的步骤

  1. 使用条件
    该元素是使用了 v-if,v-show 等让元素隐藏或显示的指令
    Vue 在插入、更新或者移除 DOM 时，提供多种不同方式的应用过渡效果

  2. 使用 6 个特别的 class 来定制动画

    类似于 a 链接的 4 个状态

    入场：一个元素从隐藏到显示。有 3 个状态
    v-enter 准备入场（隐藏）0%

    v-enter-active 入场的过程 1%-99%

    v-enter-to 入场完毕（显示）100%

    出场：一个元素从显示到隐藏
    v-leave 准备出场

    v-leave-active 出场的过程

    v-leave-to 出场完毕

# 一组元素的过渡动画效果（列表过渡）

transition 只包含一个元素，如果要让 transition 包含多个元素，需要使用 transition-group

组件还有一个特殊之处。不仅可以进入和离开动画，还可以改变定位。

如果只是单纯使用依然会出现弹跳效果，而且此时过渡模式（out-in）并不能解决该问题，这时，需要通过 CSS 定位让他们重合一起进行动画，问题的根源依然是因为 transition 进入和离开同时发生的特性。

https://cn.vuejs.org/v2/guide/transitions.html# 列表过渡

# 总结

  1. 对单个元素使用过渡（transition 标签最终包裹的是一个元素，包括 v-if,v-else 这种）使用 transition + 过渡模式 + 给元素设置 key 属性或定位
  2. 对多个元素使用过渡（transition 标签中需要包裹多个元素，可以是只显示一个元素，其他隐藏），使用 transition-group + 定位解决 + 动态设置唯一 key 属性（该方法同样适用于单元素的过渡）

官网文档：https://cn.vuejs.org/v2/guide/transitions.html
