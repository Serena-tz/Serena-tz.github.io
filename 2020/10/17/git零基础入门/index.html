# 系统命令行工具来操作电脑

CMD–command。使用使用指令的方式来操作我们的电脑。现在还在使用命令行操作的系统：Linux 和 Unix，服务器电脑中还有存在。可视化的系统：window 和 Mac

# 可以使用以下任意一种工具来输入命令行

  1. cmd

    所有的 window 系统都支持
    打开方式：
    win+R 键–>cmd–> 回车
    缺点：最古老的命令行工具，一些新的指令不支持，比如 ls

  2. powershell
    是 win10 才有的命令行工具 - 更加强大更加好用
    好处：
    可以在任意文件夹下通过 shift + 右键单击，打开该工具，可以直接定位到该目录下，而不需要多次 Cd 才能进入目录
    以后经常要使用的方式

  3. git bash
    安装了 git 软件后才有的，学 github 安装了两款软件：github 桌面版，git 软件。
    安装后，可以在任意文件夹下右键单击，选择 git bash

    好处：
    a. 完成使用命令行操作 github 的任务
    b. 像 powerShell 那样在任意文件夹下运行以上任意指令

# cmd 小技巧

  1. 使用键盘上下键可以切换到上一条或下一条指令

  2. ctrl +C 退出未完成的指令

# 场景 4：在任意文件夹下同步到远程库

步骤：

  1. 在要提交的目录下打开命令行工具

  2. 初始化 git 项目
    git init
    作用：会在当前目录下新增一个.git 目录用来记录同步状态

  3. 按照提交三部曲进行（场景 2）
    git add .
    git commit -m “注释”

  4. 关联远程目录库
    在关联前，首先要到 github 中创建一个空的资源库，然后复制远程库地址：

    语法：git remote add origin 远程库地址
    比如：
    git remote add origin https://github.com/kouok/web19-bs-project.git

  5. 正式提交
    git push -u origin master
    可能会弹出输入框让你输入用户名，密码

  6. 后续就按场景 2 或场景 3 进行（本地更新了或远程更新了才需要按场景 2 或场景 3）

注意：在已有的资源库中不能再嵌套另外一个资源库。
