# 后端开发用什么语言

  1. java 数据库 网站 | 软件 .jsp
  2. .net 微软的 + 数据库 网站 | 软件 .aspx
  3. node.js 咱们前端也可以做后端开发 网站 | 软件
  4. php 轻巧容易学，网站 | APP | 微信小程序 mysql .php
  5. python 处理 Excel 表格 AI，大数据 .py

大部分语言都是相同的，只是写法不一样。

# PHP 环境配置

php 文件的后缀 .php

PHP 文件可以写什么：

  1. php 最大的特点是混编：即可以在 PHP 文件中嵌入前端代码
  2. 也可以写纯 PHP 代码

所有的后端语言都不能直接在浏览器中运行，都需要编译。需要安装编译软件，PHP 编译的软件：PHPstudy 或 xampp 或 wampp (他们都是集成开发环境 — 傻瓜式的编译软件，不需要配环境，直接可以使用，还附带数据库)

# 如何使用 PHPstudy

  1. 开启 Apache 服务，即可解析 PHP 文件
  2. 将 PHP 文件放到特定文件夹（服务器）你的 PHPstudy 安装目录下的 www 文件夹（该文件夹在浏览器使用 localhost 来表示）

PHP 语法更多：https://www.runoob.com/php/php-tutorial.html
