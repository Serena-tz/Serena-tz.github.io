# 今日头条小程序数据库和 json 地址

数据保存到数据库中的，里面的内容如何获取？由 PHP 文件获取，并且生成一个 json 文件

提供的 3 个 PHP 文件，是用来负责连接指定的数据库，并且提取数据库内容，生成一个 json 文件

前端拿到数据源地址就可以开始干活了。

# 准备工作

  1. 搭建好 PHP 后台。
    放置 PHP 文件到 www 目录下。

    导入数据库
    在浏览器中访问 PHP 文件，看能否返回 json 数据。

  2. 数据库两张表的解释
    type: 放栏目名称–显示菜单 - getType.php 获取该 json

    news: 具体的文章内容.

    由两个 PHP 文件来处理：

    getList.php?typeId=1002 获取指定分类的文章

    1002 指的是数据库中字段的名字。

    getDetail.php?newsId=news-1002 根据文章 ID 获取某篇文章

# 写静态页面 - 首页（列表页）

  1. 顶部导航需要左右滑动

  2. 给标签添加 data - 属性
    说明：该属性是 H5 的，可以实现自定义属性

  3. 通过列表数据的 img_type 的值 0,1,3 来确定使用哪种文章模板

    使用 wx:if 来决定用哪种结构

    0 无图模式
    1 一张图
    3 三张图
