# 服务器是什么？

我们写的代码如何让全世界的人都看得到，必须使用服务器。云端：服务器。一台远程的电脑。配置和速度和网上都要比。放运营商的机房。

# 域名服务器有谁来完成（工作）

  1. 运维工程师（需要懂一定的代码）
  2. 后台也可以做（公司比较小）
  3. 前端学习的目的是方便自己，多学一个技能（每多学一项技能，就可以少说一句求人的话）

# 域名

  1. 域名和 IP 的关系

访问一个域名，访问的是该域名所代表的那个 IP 地址。

IP 是什么？

一个 IP 对应一台上网的设备。

买服务器相当于是买了个可以专用的 IP 地址。

但是 IP 地址很难记，可以买域名，然后绑定 IP，通过域名来访问。

  1. 如何申请域名

域名和域名后缀组成。

到阿里云、百度云、腾讯云等大的互联网公司都可以注册。

  1. 域名和服务器的关系

需要绑定服务器 IP 地址，才能形成一个映射，即访问域名就是在访问你绑定的这台服务器

  1. 一个 IP 可以对应多个域名

# 服务器

买服务器相当于是买了个专用 IP 地址。买了一块服务器资源。

  1. 国内服务器（备案，打算用一年）

    备案，到工信部备案。备案了才能正常使用

  2. 国外免备案服务器

    好处：免备案。

    坏处：不稳定，访问速度慢

服务器种类：

  1. 虚拟主机

    价格便宜，一台虚拟机只能绑定一个域名，299 元一年

  2. 云服务器

    价格稍贵，可以绑定多个网站，一般一个月几十元。或一年几千元。

主机和域名绑定需要绑定。到主机管理面板将其进行绑定。

# 通过 ftp 上传代码到服务器

ftp: 是一个协议。同步线上线下代码的

ftp 上传软件是实现该协议的一个工具。

推荐工具：flashftxp–跟 github 桌面版差不多

使用该软件需要知道服务器的 ftp 的账户名密码，主机服务商会提供给你。

盗梦电影项目：https://ticket.nexttix.net/
