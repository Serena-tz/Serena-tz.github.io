# event 对象

event 对象是用来记录事件发生时的所有细节 - 黑盒子，行车记录仪。

注意：不同的事件类型的 event 对象的属性有所不同。

直接通过 console.log (event) 即可

JS 的 event 和 JQ 的 event 几乎一致。JQ 的 event 没有兼容性问题。
