# JQ 文件和版本

# 完整版：

  1. 用于开发环境中（写项目）
  2. 是正常的未压缩的源代码

# 压缩版：

  1. 用于生成环境（线上）
  2. 去掉了注释、留白、并且压缩了变化、函数名等

# 版本选择：

  1. 如果要兼容老版本浏览器（即在 PC 上），最好使用 1.12.4 版本即以下的
  2. 否则使用 3 最新版本，这样可以兼容 PC 版主流浏览器 + 移动端浏览器
  3. 各个版本间区别不是太大，基本 API 都是通用的
